# Funch
tap-ball을 더욱 재미있게 하기 위해 게임플랫폼으로 만드는 프로젝트

## 기본 기능
Left, Right 펀치 카운트 세기
펀치 세기에 따라 목표 달성하기
연속 콤보 파악하기

## 시스템 구성
디바이스 : 탭볼이 글러브에 맞는 것을 파악하기 위한 것으로 글러브로 탭볼을 치면 데이터를 보낸다.
